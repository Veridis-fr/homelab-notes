# Linux Basics

## Topics covered
- File system navigation
- File permissions
- Users and groups
- Basic process management

## Commands & notes
- `ls`, `cd`, `pwd` — navigate the file system
- `cp`, `mv`, `rm` — manage files and directories
- `chmod`, `chown` — manage file permissions and ownership
- `ps`, `top` — view running processes
- `systemctl` — manage services

## What I learned
Linux is built around permissions and processes.
Understanding how files, users, and services interact is essential
for system administration and troubleshooting.
